# Agentic AI & Information Integrity

A public research hub connecting projects, papers, datasets, and artifacts around:

- agentic AI
- synthetic media and disinformation
- human perception and cognitive vulnerability
- provenance and verification
- information integrity on the Web

This repository is the **top-level entry point** for a broader research program. It does **not** duplicate the individual project repositories. Instead, it provides:

- an overview of how the projects fit together
- a curated index of research artifacts
- mobile-friendly landing pages in `docs/`
- a stable QR-friendly destination for conference materials

## Research Program

### Core Projects

#### JudgeGPT
An empirical platform for measuring how humans judge AI-generated news on dual axes: source attribution and authenticity.

- Repo: <https://github.com/aloth/JudgeGPT>
- Demo: <https://judgegpt.streamlit.app/>
- Preprints: <https://arxiv.org/abs/2601.21963>, <https://arxiv.org/abs/2601.22871>, <https://arxiv.org/abs/2404.03021>
- Zenodo: <https://doi.org/10.5281/zenodo.18703385>

#### RogueGPT
A controlled stimulus generation engine for synthetic news and misinformation research.

- Repo: <https://github.com/aloth/RogueGPT>
- Preprints: <https://arxiv.org/abs/2601.21963>, <https://arxiv.org/abs/2601.22871>, <https://arxiv.org/abs/2404.03021>
- Zenodo: <https://doi.org/10.5281/zenodo.18703138>

#### Origin Lens
A privacy-first mobile framework for on-device image provenance verification using C2PA and related signals.

- Repo: <https://github.com/aloth/origin-lens>
- App: <https://apps.apple.com/us/app/origin-lens/id6756628121?platform=iphone>
- Preprint: <https://arxiv.org/abs/2602.03423>

#### verification-crisis
An expert survey on GenAI disinformation threats, reproducible provenance, and mitigation strategy preferences.

- Repo: <https://github.com/aloth/verification-crisis>
- Survey: <https://forms.gle/BCwYFtfqxmZewkL97>
- Preprint: <https://arxiv.org/abs/2602.02100>
- Zenodo: <https://doi.org/10.5281/zenodo.18703601>

#### CRED-1
A credibility-oriented dataset and infrastructure layer for source-level information integrity research.

- Repo: <https://github.com/aloth/cred-1>
- Dataset: <https://doi.org/10.5281/zenodo.18769460>
- Preprint / paper page: <https://doi.org/10.2139/ssrn.6448466>
